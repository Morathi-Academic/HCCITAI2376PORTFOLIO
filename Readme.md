### 2. **VGG16 Pretrained Model Lab**
**Folder:** `VGG16_Lab/`

This lab focuses on applying the VGG16 pretrained CNN model (trained on ImageNet) to perform image classification.  
Key concepts explored:
- Transfer learning  
- Feature extraction  
- Understanding model behavior with altered inputs  
- Effects of rotation, noise, and distortions  
- Interpreting model confidence  

This lab builds intuition about deep neural networks and how they interpret image data.

### 3. **Diffusion Model ‚Äì Image Generation**
**Folder:** `Diffusion_Model/`

This assignment implements a simplified diffusion model to generate synthetic MNIST-style images starting from pure noise.  
Core learning areas:
- Forward diffusion (adding noise)  
- Reverse denoising steps  
- Sampling from a trained model  
- Visualizing the generative process  
- Understanding modern generative AI foundations  

This project introduces the concepts behind stable diffusion and modern generative models.
